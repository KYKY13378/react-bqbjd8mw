# Тренажёр слов - React приложение

## Описание
Приложение для изучения английских слов с возможностью просмотра перевода и перехода к следующему слову.

## Функциональность
- Показ случайного английского слова
- Отображение перевода по кнопке
- Возможность перехода к следующему слову
- Адаптивный дизайн

## Технологии
- React
- CSS3
- HTML5

## Установка и запуск
1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Запустить приложение: `npm start`
